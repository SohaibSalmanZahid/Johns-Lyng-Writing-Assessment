# Johns-Lyng-Writing-Assessment

A simple Todo App where the user is able to see their TODO list, add items to it and delete items from it

# Development Server

The application runs on a fixed port: 4200

To change the port:

1. Open `angular.json`
2. Navigate to `projects > ToDo-Application > architect > serve > configurations > development`
3. Modify the `port` value
4. Update backend CORS/API configuration to allow the new port
